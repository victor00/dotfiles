# dotfiles


```bash
.dotfiles/
├── bin/.local/scripts/
├── git/
│   └── .gitconfig
├── zsh/
│   └── .zshrc
├── tmux/
│   └── .tmux.conf
├── vim/
│   └── init.vim
└── i3/
    └── config
```


### Run script to setup the machine

```bash
chmod +x ~/dotfiles/bin/.local/scripts/main.sh

~/dotfiles/bin/.local/scripts/main.sh
```
